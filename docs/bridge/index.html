Stock Manager

Puente BLE — Modo portátil

Lee la báscula Bluetooth y envía los pesos a tu Home Assistant desde cualquier lado. Necesita Chrome en Android (Web Bluetooth no existe en iOS).
Configuración …
Tu URL HTTPS de HA — la que ya usás para acceder desde fuera de casa.
HA → Perfil → "Tokens de acceso de larga duración" → Crear token. Pegalo entero.
Solo se usa si la báscula no expone su ID por BLE. Es el ID numérico de la báscula en la addon.
Lecturas por segundo enviadas a HA. La báscula notifica más rápido; el puente filtra para no saturar.
Báscula Desconectado
Tocá "Conectar báscula" y elegí tu dispositivo (algo tipo Stock-Scale-…).
